You are a senior frontend engineer and product designer.
Build a complete React-based web application called “StuDENt” – a professional student productivity and tools platform.
PROJECT OVERVIEW
StuDENt is an all-in-one tools website for school and college students.
It provides 39 free tools in V1 with no login and no subscription.
Target users:
- School students
- College / university students
Core principles:
- Clean, modern, professional UI
- Fast, frontend-only
- Real logical workflows (not fake buttons)
- Honest V1 feature scope
TECH STACK
- React (Vite or CRA)
- Tailwind CSS (preferred) or modern CSS
- shadcn/ui–style components
- Frontend-only (NO backend)
- Browser APIs and JS logic only
GLOBAL RULES (STRICT)
- Build everything from scratch
- No authentication
- No backend
- No paid or external APIs
- Use browser-based libraries only
- All tools must open, accept input, process, and show output
- No “Coming Soon” text anywhere

WORKFLOW REQUIREMENTS (FOR EVERY TOOL)
Each tool must follow this flow:
1. User opens tool
2. User provides input (text, number, or file)
3. User clicks action button
4. Show loading animation (1–3 seconds)
5. Process input logically (real JS or browser logic)
6. Show output/result
7. Allow copy or download where applicable
No dead buttons. No empty pages.

- Searches tool name + keywords
- Results update while typing
